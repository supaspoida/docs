## Creating a UiHook

The UiHook we will create is a simple counter. It's a good example for understanding how to create your first ZEIT addon.

Create an `index.js` file and add the following code:

```js
const { withUiHook } = require('@zeit/addons')

let count = 0

module.exports = withUiHook(({ payload }) => {
  count += 1
  return `
 		<Page>
 			<P>Counter: ${count}</P>
 			<Button>Count Me</Button>
        </Page>
 	`
})
```

<Caption>

The code is simple. Inside the UiHook, we increment the count and send JSX response with the current count.

### Running the UiHook Locally

With the [help of `now dev`](/docs/v2/development/basics), you can run your UiHook's locally inside a simulation of the Now platform.

To run the UiHook locally, use the following command:

<TerminalInput>now dev -p 5005</TerminalInput>
